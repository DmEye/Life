О коде
--------------
Переменные и списки
- Список field хранит в себе значения с нулями и единицами, где нуль - клетка мертва, а единица - жива.
- Он же используется для отрисовки.

- Список newField нужен для расчёта следующего поколения.

- Переменная n отвечает за ширину поля.

- Переменная live нужна для управления главного цикла

- Переменная score отслеживает кол-во поколений

Процедуры и функции
- Процедура create устанавливает ширину полей field и newField
- А так же она заполняет поля нулями

- Процедура fill заполняет поле field единицами с учётом указанной вероятности

- Процедура draw перерисовывает поле field

- Функция neighbours подсчитывает кол-во "живых" клеток вокруг указанной клетки  поля field

- Процедура generation расчитывает следующее поколение

- Процедура logic нужна для определения конца игры

- Процедура glider создаёт на указанном поле в указанных координатах "планер" (подробнее на википедии https://ru.wikipedia.org/wiki/%D0%98%D0%B3%D1%80%D0%B0_%C2%AB%D0%96%D0%B8%D0%B7%D0%BD%D1%8C%C2%BB)
